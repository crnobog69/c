Satri Programming Language

Satri is a programming language with Serbian-inspired syntax that supports basic programming constructs, character operations, and includes unique features like Blood Meridian easter eggs.

Installation

gcc -o satri main.c lexer.c interpreter.c

Basic Syntax

Variables and Data Types

// Integer variables
int x, y, z;
x = 42;

// Character variables
char c;
c = 'A';

// Boolean variables
bool istina = tacno;    // true
bool laz = netacno;     // false

Input/Output

// Output
ispis = "Zdravo svete!";    // Print string
ispis x;                    // Print variable

// Input
ucitaj x;                   // Read input into variable

Functions

def saberi {
    int a, b;
    a = 10;
    b = 20;
    ispis a + b;
}

// Call function
saberi;

Character Operations

char c;
c = 'A';

u_broj c;      // Convert to ASCII number
veliko c;      // Convert to uppercase
malo c;        // Convert to lowercase
je_broj c;     // Check if digit
je_slovo c;    // Check if letter

Easter Eggs

Satri includes two special commands that print quotes from Blood Meridian: 

bloodmeridian;    // Prints English quote
krvavimeridijan;  // Prints Serbian quote

Comments

// Single line comment

/*
   Multi-line
   comment
*/

File Extension

Satri programs use the .st file extension.

Program Termination

Every program must end with the `judge` keyword.

Use the `judge` keyword to terminate the program.

Running the program

To run the program, use the following command: ./satri program.st

Početni Primeri (Beginner Examples)

1. Prvi Program (First Program)

def main {
    stampaj = "Zdravo svete!";
}

main;
judge

2. Rad sa Brojevima (Working with Numbers)

def saberi {
    broj x = 5;
    broj y = 3;
    stampaj x + y;  // Ispisuje 8
}

saberi;
judge

3. Unos i Ispis (Input and Output)

def program {
    broj x;
    stampaj = "Unesite broj:";
    ucitaj x;
    stampaj = "Uneli ste:";
    stampaj x;
}

program;
judge
